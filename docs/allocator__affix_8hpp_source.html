19 /* Affix Allocator: allocate(sizeof(Prefix) + size + sizeof(Suffix))
20  * usefull for debug, statistics, information, ...
21  * use Prefix & Suffix as fences to detect wrong writes: "BBBBBBBB"..."EEEEEEEE"
22  * use Prefix & Suffix for human readable text to ease memory reading: "type=MyClass"..."end of type=MyClass"
23  * use Prefix & Suffix to store information & statistics (creation timestamp, source code file & line, access count...)
