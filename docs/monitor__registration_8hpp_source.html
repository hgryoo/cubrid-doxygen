21 // monitor_registration.hpp - interface for cubrid monitor and statistic registration
22 //
23 // this interface defines the cubrid statistics monitor and how to register statistics.
24 //
25 // all statistics that should be inspected by cubrid statdump tool should be registered to performance monitoring.
26 // to register a statistic, one must provide:
27 //
28 // 1. statistic name

30 // 3. TODO: other statistic properties.
