43 /*
44  * This is NOT the "object" class but rather functions more like
45  * the meta-class of class objects.
46  * This formerly stored the list of classes that had no super classes,
47  * in that way it was kind of like the root "object" of the class
48  * hierarchy. Unfortunately, maintaining this list caused contention
49  * problems on the the root object so it was removed. The list
50  * of base classes is now generated manually by examining all classes.
51  */
